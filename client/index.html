About
Modulo analyses NPM modules to find the most up-to-date, stable, and widely-used libraries to fit the user's needs. Module quality metrics are exposed to the user immediately upon searching, and advanced statistics are visualized for in-depth comparisons.

Built with Angular, Neo4j, Express and Node. Copyright 2015. Contact us.
